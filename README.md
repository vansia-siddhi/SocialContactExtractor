# Social Contact Extractor

A professional Python application to extract contact details (phone, email, office address) from social media profiles.

## Features

- 🚀 Extract contacts from LinkedIn, Instagram, Facebook, Twitter, YouTube
- 📱 Professional responsive UI
- 🔒 Secure and private
- 📊 Real-time extraction
- 🎯 Platform detection
